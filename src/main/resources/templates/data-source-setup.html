Data Source & Calculation Setup

Data Source Configuration

Objective

Configure primary data sources and calculation methodologies for each facility and emission type. Ensure all relevant emission factors are properly assigned to provide accurate and consistent GHG calculations.

Data Collection Methods

Configure how activity data is collected for each emission source:

  • Utility bills & invoices
  • Direct metering
  • ERP system integration
  • Engineering estimates
Emission Factors

Apply standardized conversion factors appropriate for each activity:

  • Location-specific factors
  • Energy & fuel factors
  • Transportation factors
  • Historical tracking
Calculation Methods

Select appropriate calculation approaches for different emissions:

  • Activity-based calculations
  • Mass balance approach
  • Process-specific models
  • Compliance-approved methods
